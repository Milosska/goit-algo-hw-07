# Інструкція щодо використання репозиторію

1. Склонуйте репозиторій

```bash
git clone [repository-url]
cd goit-algo-hw-06
```

2. Перейдіть у директорію проєкту та активуйте віртуальне середовище

```bash
 python -m venv .venv
 # On Windows:
 .venv\Scripts\activate
 # On Unix/MacOS:
 source .venv/bin/activate
```

3. Встановіть необхідні залежності

```bash
# Install all dependencies (including development tools)
pip install -r requirements.txt
```

4. Запустіть головний файл main.py в папці з потрібним завданням

```bash
python main_task\main.py
```

## Основне завдання (завдання 1-3)

Взято базову реалізацію AVL-дерева. До базової реалізації додано наступні функції:

- `min_value_node` - повертає вузол дерева з найменшим значенням ключа
- `max_value_node` - повертає вузол дерева з найбільшим значенням ключа
- `get_total` - повертає суму значень ключів всіх вузлів дерева

## Додаткове завдання

Реалізовано структуру даних для системи коментарів так, щоб коментарі могли мати відповіді, які, в свою чергу, також могли мати відповіді, формуючи таким чином ієрархічну структуру.

Реалізовано клас Comment, що представляє собою окремий коментар. Він зберігає текст коментаря, автора та список відповідей.

Реалізовано наступні методи класу:

- Метод класу add_reply, що додає нову відповідь до коментаря.
- Метод класу remove_reply, який "видаляє" відповідь із коментаря, змінюючи змінювати текст коментаря на стандартне повідомлення "Цей коментар було видалено".
- Метод display рекурсивно виводить коментар та всі його відповіді, використовуючи відступи для відображення ієрархічної структури.
